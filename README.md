`Shopping Cart Notifications` module allows to add additional pop-up notification for the customer on the top of the screen about adding a product to cart. ![Notification](http://storage1.static.itmages.com/i/16/0106/h_1452075938_9758339_3faf6de694.png "Notification")

Also, this module has the ability to show the overlay animated loader while the product is adding to the shopping cart.

Settings of the module are located in 
Admin Panel->Stores->Configuration->MageWorkshop->Shopping Cart Notifications.

Available settings:
* overlay settings;
* time of notification displaying;
* notification settings.
